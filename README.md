### Customer Segmentation & Data Mining — Food Delivery Platform

Analyzed a large-scale real-world food delivery dataset with **31K users and 50+ features** to identify meaningful customer segments and support data-driven marketing strategies.

**Key work included:**

- Engineered structured features from spending behavior, cuisine preferences, ordering patterns, and vendor interactions

- Selected the optimal number of clusters using elbow method and silhouette analysis
- Reduced redundant features using correlation-based feature selection

**Tools:** Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn  
**Focus Areas:** Customer Segmentation, Data Mining, Clustering, Business Analytics

### Workers’ Compensation Claim Classification

Built a multiclass machine learning pipeline to classify workers’ compensation claim injury types using a dataset of **593,471 records and 33 features**.

**Key work included:**
- Engineered domain-specific features from demographic, injury, and missingness indicators
- Standardized missing values and filtered anomalous records
- Compared Logistic Regression, Random Forest, and XGBoost models
- Used repeated 6-fold cross-validation with macro F1-score
- Selected XGBoost as the final model based on generalization performance

**Tools:** Python, Pandas, NumPy, Scikit-learn, XGBoost, Matplotlib, Seaborn  
**Focus Areas:** Classification, Feature Engineering, Model Evaluation, Predictive Analytics
